## Instalação

OS X & Linux:

```sh
yarn

# ou caso use npm

npm install
```

## Iniciando a aplicação web

OS X & Linux:

```sh
yarn start

# ou caso use npm

npm start
```

Em seguida abra o seguinte link no seu browser : http://localhost:3000
